# boilerplate
Boilerplate intended for server-less applications, but can be used for applications with servers.

Usage:

To Build

 1. `git clone https://github.com/byshine/boilerplate.git`
 2. `npm install`
 3. `npm run build`
 
 For development
 
 1. `git clone https://github.com/byshine/boilerplate.git`
 2. `npm install`
 3. `npm run develop`
